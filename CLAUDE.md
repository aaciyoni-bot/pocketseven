# Project memory — PocketNine (ORIZIS TECHNOLOGY private label)

## Brand rules for this repo
- PocketNine is a PRIVATE LABEL: it must never visually resemble POKERTEN or
  carry ORIZIS branding. By explicit owner decision this product does NOT
  open with the ORIZIS intro — it opens with the PocketNine "printing the 9"
  intro (four CMYK plates stamping on paper) embedded in index.html.
- Design language: "The Press Room" — paper #f3f2f2, ink #201e1d, cyan
  #0088b0 for interaction, magenta #d6006c as rare spot, Source Serif 4
  everywhere, 2px radius, no gradients/glows. See the pn-* tokens in
  index.html.
- Functionality is kept in lockstep with the pokerten repo (same engine,
  same server functions); only the skin differs.

Owner: aaci.yoni@gmail.com (ORIZIS TECHNOLOGY).
