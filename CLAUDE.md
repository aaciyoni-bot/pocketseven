# Project memory — Pocket Seven (ORIZIS TECHNOLOGY private label)

## Brand rules for this repo
- Pocket Seven is a PRIVATE LABEL: it must never carry ORIZIS branding. By
  explicit owner decision this product does NOT open with the ORIZIS intro —
  it opens with the Pocket Seven intro (gold wordmark on a bordeaux-black
  stage) embedded in index.html.
- Identity comes from the extracted Pocket Seven brand assets (APK design
  handoff): logo `logo-wordmark.png` (POCKET / SE7EN, white with a gold 7),
  motif = the number 7.
- Design language: premium dark casino. Tokens are the `--ps-*` set in
  index.html:
  - grounds bordeaux-black (#0C0908 / #140D0C), panels #1C1211 / #271010
  - accent metallic gold #F8C830 (deep #D89830, light #F8D030) — the only
    accent; the 7 in the logo, buttons, highlights
  - green felt #217D4E / #3C8A56, victory blue #2E5C8F
  - type: Cinzel (display/brand/headings) + Inter (UI/numbers) +
    Noto Sans Hebrew (RTL). No Source Serif / no "Press Room" paper theme.
- Functionality is kept in lockstep with the pokerten repo (same engine,
  same server functions); only the skin differs.
- Firebase project id `pocketnine-51af8` is internal and stays as-is; the
  user-facing product name and domain are Pocket Seven / pocketseven.com.
